## üöÄ Quick Install Guide

### Install C/C++ (Most Important Next Step)
```powershell
# Option 1: w64devkit (Recommended - All-in-one)
# Download from: https://github.com/skeeto/w64devkit/releases
# Extract and add to PATH

# Option 2: MinGW-w64
# Download from: https://www.mingw-w64.org/
# Run installer and add bin folder to PATH
```

### Install Go
```powershell
winget install GoLang.Go
# Or download from: https://go.dev/dl/
```

### Install Rust
```powershell
# Download and run: https://rustup.rs/
# Or use: winget install Rustlang.Rustup
```

### Install Ruby
```powershell
winget install RubyInstallerTeam.Ruby.3.2
```

### Install PHP
```powershell
winget install PHP.PHP
```

### Verify Installation
```powershell
gcc --version      # C
g++ --version      # C++
go version         # Go
rustc --version    # Rust
ruby --version     # Ruby
php --version      # PHP
```
