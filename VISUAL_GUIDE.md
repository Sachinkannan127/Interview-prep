## Color Scheme Reference

### Score Badges:
- **Excellent:** `bg-green-100 text-green-700`
- **Good:** `bg-blue-100 text-blue-700`
- **Fair:** `bg-yellow-100 text-yellow-700`
- **Needs Improvement:** `bg-red-100 text-red-700`

### Avatar:
- **Background:** Gradient from slate-800 to slate-900
- **Avatar Circle:** Gradient from primary-400 to purple-500
- **Speaking Indicator:** Green-500
- **Waveform:** Primary-400
- **Status Light:** Green-500 (active) / Slate-500 (inactive)

### Buttons:
- **Primary:** Gradient indigo/purple
- **Secondary:** Ghost style with hover
- **Regenerate:** Spinning icon on click

## Animation Details

### Question Regeneration:
- Click ğŸ”„ â†’ Icon spins â†’ Question fades out â†’ New question fades in
- Duration: ~1-2 seconds

### Avatar Animations:
- **Speaking:** Mouth pulses, waveform animates, circle scales up 10%
- **Listening:** Neutral mouth, no waveform, normal scale
- **Thinking:** Eyes squint, mouth neutral

### Waveform:
- 20 vertical bars
- Each bar height: 10-50px (random)
- Animation: Wave pattern with 0.3-0.8s timing
- Staggered delay: 0.05s per bar

### Status Transitions:
- Smooth fade between states
- 300ms transition duration
- Scale transform on state change

## Accessibility Features

### Keyboard Navigation:
- **Tab:** Navigate between questions and buttons
- **Enter/Space:** Activate regenerate buttons
- **Escape:** Close avatar (if focused)

### Screen Reader Support:
- Avatar states announced: "AI is speaking", "Listening"
- Score labels read with descriptions
- Question numbers announced properly

### Color Contrast:
- All text meets WCAG AA standards
- Score badges have sufficient contrast
- Avatar elements clearly visible

---

## Performance Considerations

### Question Generation:
- API call: ~2-5 seconds for 5 questions
- Individual regeneration: ~1-2 seconds
- Caching: Questions stored in state (no re-fetch)

### Avatar Rendering:
- requestAnimationFrame for smooth 60fps
- Conditional rendering (only when enabled)
- Cleanup on unmount to prevent memory leaks

### Score Calculations:
- Client-side utility functions (instant)
- No additional API calls needed
- Memoized where appropriate
